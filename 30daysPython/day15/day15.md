#  Day 15 - Notes Manager

##  Project Overview

A command-line Notes Manager built using Python that allows users to create, organize, search, update, and delete notes. Notes are stored permanently using a JSON file.

---

##  Features

- ➕ Add new notes
- 📖 View all notes
- 🔍 Search notes by title
- 🏷 Search notes using tags
- ✏ Update existing notes
- ❌ Delete notes
- 💾 JSON file storage
- 🧩 Object-Oriented Programming (Classes)
- 📂 File Handling
- 📋 CRUD Operations

---

##  Technologies Used

- Python 3
- JSON
- File Handling
- OOP (Classes & Objects)
