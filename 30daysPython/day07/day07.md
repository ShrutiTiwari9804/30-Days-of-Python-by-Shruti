#  Expense Tracker (Day 07)

##  Project Overview

Expense Tracker is a simple command-line Python application that helps users record and manage daily expenses.

The application stores expense records in a text file so that data remains available even after closing the program.

---

##  Features

- Add new expenses
- View all expenses
- Calculate total spending
- Search expenses by category
- Persistent storage using text files
- Menu-driven interface
- Input validation

---

##  Technologies Used

- Python 3
- File Handling
- Functions
- Loops
- Exception Handling
- String Formatting
