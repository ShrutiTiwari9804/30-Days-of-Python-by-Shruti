# 🛒 Day 14 - Shopping Cart System

## Overview

This project is a console-based Shopping Cart System developed using Python classes and Object-Oriented Programming (OOP).

Users can add products, update quantities, remove products, and calculate the total bill. The project demonstrates CRUD operations using classes.

---

## Features

- Add Product
- View Cart
- Update Product Quantity
- Remove Product
- Calculate Total Bill
- Object-Oriented Programming
- CRUD Operations

---

## Technologies

- Python 3

---

## Concepts Used

- Classes & Objects
- Constructors (__init__)
- Methods
- Lists
- CRUD Operations
- Encapsulation
- Loops
- Conditional Statements

| Operation | Function |
