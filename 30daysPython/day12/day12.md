# 🎬 Day 12 – Movie Collection Manager

## Project Overview

Movie Collection Manager is a command-line Python application that helps users manage their personal movie collection.

Users can store movie information, search titles, update details, delete records, mark movies as watched, and assign ratings.

The project uses JSON as a simple database, making it an excellent introduction to backend CRUD operations.

---

## Features

- Add movies
- Search movies
- Update details
- Delete movies
- Give ratings
- Mark watched
- Filter watched movies
- Persistent JSON storage

---

## Concepts Used

- JSON
- File Handling
- CRUD
- Functions
- Dictionaries
- Loops
- Error Handling
