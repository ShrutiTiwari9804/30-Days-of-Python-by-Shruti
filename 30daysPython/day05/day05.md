# 📚 Day 5 - Student Record Management System

## Overview
A simple command-line application that allows users to manage student records.

## Features

- Add Student
- Display Students
- Search Student
- Update Marks
- Delete Student
- Input Validation using Exception Handling

## Concepts Practiced

- Functions
- Lists
- Dictionaries
- Loops
- Conditional Statements
- Exception Handling
- CRUD Operations

## Sample Menu

## Learning Outcome

This project teaches how real applications store and manage data using Python collections and functions.
