# Personal Profile & BMI Calculator.

## Overview
This project is a simple command-line application that collects user information, performs calculations, and displays a formatted profile.

## Topics covered
- Variables
- Data Types
- User Input
- Type casting
- Arithmetic Operators
- f-String
- 'type()' Function
- Mathematical Calculations
- 'round()' Function

## Concepts Practiced

- 'input()'
- 'int()'
- 'float()'
- 'str()'
- Arithmetic operations 
- Formatted printing using f-strings
- Variable assignment


## Features
- Takes user details as input.
- Calculate BMI.
- Displays formatted profile.
- Shows Python data types.
- Demonstrates type conversion.
