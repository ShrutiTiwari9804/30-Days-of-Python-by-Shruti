# 📦 Day 16 - Inventory Management System

## 📌 Project Overview

A command-line Inventory Management System developed using Python and Object-Oriented Programming (OOP). The application allows users to manage products with complete CRUD operations while storing data permanently using JSON.

---

## 🚀 Features

- ➕ Add New Product
- 📋 View All Products
- 🔍 Search Product by Name
- ✏ Update Product Details
- ❌ Delete Product
- ⚠ Low Stock Report
- 💰 Calculate Total Inventory Value
- 💾 JSON File Storage
- 🧩 Object-Oriented Programming

---

## 🛠 Technologies Used

- Python 3
- JSON
- OOP (Classes & Objects)
- File Handling
