# Day 08 - Contact Book

## Project Overview

A command-line Contact Book built with Python.

This project performs CRUD operations:
- Add Contact
- View Contacts
- Search Contact
- Update Contact
- Delete Contact

Contacts are stored permanently using a JSON file.

## Concepts Used

- Functions
- Lists
- Dictionaries
- JSON
- File Handling
- Loops
- Conditionals
- CRUD Operations

## Skills Learned

- Working with structured data
- Reading and writing JSON
- Data persistence
- Organizing larger Python programs
