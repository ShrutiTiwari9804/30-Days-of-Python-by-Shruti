# 🏥 Day 17 - Hospital Appointment System

## 📌 Project Overview

A command-line Hospital Appointment System developed using Object-Oriented Programming (OOP) in Python. The project demonstrates how multiple classes interact using object relationships to manage patients, doctors, and appointments.

---

## 🚀 Features

- 👤 Register Patients
- 👨‍⚕️ Add Doctors
- 📋 View Registered Patients
- 🩺 View Available Doctors
- 📅 Book Appointments
- 📖 View All Appointments
- 🧩 Demonstrates OOP Relationships
- 📂 Menu-Driven Interface

---

## 🛠 Technologies Used

- Python 3
- Object-Oriented Programming (OOP)

## 📚 OOP Concepts Used

### Classes

- Patient
- Doctor
- Appointment
- Hospital

### Relationships

**Association**

- Appointment connects a Patient and a Doctor.

**Composition**

- Hospital contains Patients, Doctors, and Appointments.

### Objects

Every patient, doctor, and appointment is created as an object.
