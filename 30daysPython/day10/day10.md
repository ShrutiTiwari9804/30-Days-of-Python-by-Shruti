# Day 10 - Log File Analyzer

## Project Overview

A Python tool that analyzes application log files and generates a summary.

### Features
- Read log files
- Count ERROR, WARNING and INFO messages
- Find the most frequent error
- Generate a report
- Save report to a text file

## Concepts Practiced
- File Handling
- Exception Handling
- Dictionaries
- Collections.Counter
- String Processing

## Future Improvements
- Analyze multiple log files
- Export reports as CSV
- Visualize results using Matplotlib
- Add date-based filtering

## Why this project?

Log analysis is a common task for backend developers, DevOps engineers, and system administrators. It helps identify bugs, monitor application health, and troubleshoot issues.
