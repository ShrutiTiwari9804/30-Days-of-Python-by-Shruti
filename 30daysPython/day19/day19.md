# 🍔 Food Ordering System (Day 19)

## 📌 Project Overview

This is a console-based Food Ordering System developed using Python.

Users can:
- Add food items
- Display the menu
- Order food
- View their cart
- Remove food from the cart
- Calculate the total bill

---

## 🛠 Technologies Used

- Python 3
- Object-Oriented Programming (OOP)

---

## 📚 Concepts Covered

- Classes and Objects
- Constructors (`__init__`)
- Lists
- Functions
- Loops
- Conditional Statements
- User Input
- Object Methods
- Basic Billing Logic

---

## 📂 Features

### 1. Add Food
Allows the user to add new food items.

### 2. Display Menu
Displays all available food items.

### 3. Order Food
Lets the user add food items to their cart.

### 4. View Cart
Displays all ordered items along with the total bill.

### 5. Remove From Cart
Removes an item from the cart.

### 6. Exit
Closes the application.

---

```bash
python day19.py
```
