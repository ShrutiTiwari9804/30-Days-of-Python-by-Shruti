# 📁 Day 11 – Automatic File Organizer

## 📌 Project Overview

This Python project automatically organizes files into folders based on their file extensions.

Instead of manually sorting files, this script scans a folder and creates categories such as:

- Images
- Documents
- Music
- Videos
- Python Files
- Archives
- Others

Every file is moved to its respective folder automatically.

---

## 🚀 Features

✔ Automatically detects file types

✔ Creates folders if they don't exist

✔ Supports multiple extensions

✔ Handles unknown file types

✔ Beginner-friendly automation project
