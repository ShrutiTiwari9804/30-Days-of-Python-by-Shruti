# Day 04 - Student Report Card Generator

## Project Overview

This is Day 04 of my **30 Days Python Challenge**.

In this project, I built a Student Report Card Generator that accepts marks for multiple subjects, calculates the average, assigns grades, and displays a formatted report card.

## Features
- Accepts student details.
- Takes marks for five subjects.
- Validates user input.
- Calculates average marks.
- Automatically assigns grade.
- Displays a professional report card.

## Concepts Used


- Functions 
- Tuples
- Lists
- Dictionaries
- Loops
- Exception Handling
- String Formatting 
- Conditional Statements
- Average Calculations

## What I Learned

- Calculating averages.
