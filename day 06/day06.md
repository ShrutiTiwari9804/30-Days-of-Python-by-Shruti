# Day 6 - Password Strength Analyzer

## Project Overview

This project checks the strength of a password using multiple security rules.

## Concepts Used

- Functions
- Strings
- Loops
- Boolean values
- any()
- Character methods
- Code organization

## Password Rules

✔ At least 8 characters

✔ One uppercase letter

✔ One lowercase letter

✔ One number

✔ One special character

## Example

Enter your password:

Python@123

Output

Strength : Very Strong

Checklist

Minimum 8 characters : True

Uppercase : True

Lowercase : True

Number : True

Special Character : True
